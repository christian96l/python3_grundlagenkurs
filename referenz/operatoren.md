
### Operatoren

Die Rechenzeichen zwischen den Zahlen nennt man **Operatoren**. Python kennt sieben arithmetische Operatoren:

| Operator | Beschreibung |
|----------|--------------|
| `+`      | Addition |
| `-`      | Subtraktion |
| `*`      | Multiplikation |
| `/`      | Division |
| `//`      | Division mit Abrunden |
| `%`      | Modulo (Rest der Division) |
| `**`      | Potenz |
